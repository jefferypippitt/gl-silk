### API Reference

| Prop            | Type                                                            | Default          | Description                                |
| --------------- | --------------------------------------------------------------- | ---------------- | ------------------------------------------ |
| `variant`       | `"default" \| "outline" \| "secondary" \| "ghost"`              | `"default"`      | Button variant (same as shadcn Button)     |
| `size`          | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"` | `"default"`      | Button size variant                        |
| `duration`      | `number`                                                        | `0.5`            | Animation duration in seconds              |
| `ease`          | `string`                                                        | `"power2.inOut"` | GSAP easing function                       |
| `asChild`       | `boolean`                                                       | `false`          | Render as child component using Radix Slot |
| `showText`      | `boolean`                                                       | `false`          | Show text alongside the icon               |
| `playText`      | `string`                                                        | `"Play"`         | Text to display when paused                |
| `pauseText`     | `string`                                                        | `"Pause"`        | Text to display when playing               |
| `onStateChange` | `(isPlaying: boolean) => void`                                  | â€”                | Callback when state changes                |
