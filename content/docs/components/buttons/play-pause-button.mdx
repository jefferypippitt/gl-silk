---
title: Play Pause Button
description: Button with morphing SVG animation between Play and Pause icons using GSAP
---

<Steps>
<Step>
### Install the GSAP dependency

```npm
npm install gsap
```

</Step>
<Step>
### Add the component

```npm
npx shadcn@latest add https://glsilk.vercel.app/r/play-pause-button.json
```

</Step>
</Steps>

## Quick Start

```tsx title="app/page.tsx"
import { PlayPauseButton } from "@/components/ui/play-pause-button";

export default function Home() {
  return (
    <div>
      <PlayPauseButton />
    </div>
  );
}
```
