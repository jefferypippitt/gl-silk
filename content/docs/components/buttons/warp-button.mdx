### Advanced Text Effects

For advanced text effects, use the `textClassName` prop to add custom styling:

```tsx
// Add mix-blend-difference for color inversion effect
<WarpButton textClassName="text-white mix-blend-difference">
  Inverted Text
</WarpButton>
```

### API Reference

| Prop              | Type                                                                                               | Default          | Description                                |
| ----------------- | -------------------------------------------------------------------------------------------------- | ---------------- | ------------------------------------------ |
| `variant`         | `"default" \| "destructive" \| "destructive-red" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"default"`      | Button variant (same as shadcn Button)     |
| `size`            | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"`                                    | `"default"`      | Button size variant                        |
| `asChild`         | `boolean`                                                                                          | `false`          | Render as child component using Radix Slot |
| `colors`          | `string[]`                                                                                         | Default gradient | Array of colors for the warp effect        |
| `proportion`      | `number`                                                                                           | `0.45`           | Proportion of the warp effect              |
| `softness`        | `number`                                                                                           | `1`              | Softness of the warp edges                 |
| `distortion`      | `number`                                                                                           | `0.25`           | Amount of distortion applied               |
| `swirl`           | `number`                                                                                           | `0.8`            | Intensity of the swirl effect              |
| `swirlIterations` | `number`                                                                                           | `10`             | Number of swirl iterations                 |
| `shape`           | `"checks" \| "stripes" \| "edge"`                                                                  | `"checks"`       | Shape pattern for the warp                 |
| `shapeScale`      | `number`                                                                                           | `0.1`            | Scale of the shape pattern                 |
| `speed`           | `number`                                                                                           | `1`              | Animation speed                            |
| `frame`           | `number`                                                                                           | —                | Frame number for animation control         |
| `scale`           | `number`                                                                                           | `1`              | Scale of the warp effect                   |
| `rotation`        | `number`                                                                                           | —                | Rotation of the warp effect                |
| `offsetX`         | `number`                                                                                           | —                | X offset for the warp effect               |
| `offsetY`         | `number`                                                                                           | —                | Y offset for the warp effect               |
| `width`           | `number \| string`                                                                                 | —                | Width of the warp effect                   |
| `height`          | `number \| string`                                                                                 | —                | Height of the warp effect                  |
| `fit`             | `"contain" \| "cover"`                                                                             | `"cover"`        | How the warp effect fits                   |
| `worldWidth`      | `number`                                                                                           | —                | World width for the warp effect            |
| `worldHeight`     | `number`                                                                                           | —                | World height for the warp effect           |
| `originX`         | `number`                                                                                           | —                | X origin for the warp effect               |
| `originY`         | `number`                                                                                           | —                | Y origin for the warp effect               |
| `minPixelRatio`   | `number`                                                                                           | —                | Minimum pixel ratio                        |
| `maxPixelCount`   | `number`                                                                                           | —                | Maximum pixel count                        |
| `className`       | `string`                                                                                           | —                | Additional CSS classes for the button      |
| `textClassName`   | `string`                                                                                           | —                | Additional CSS classes for the text        |
