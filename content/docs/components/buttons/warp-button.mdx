---
title: Warp Button
description: Animated button with dynamic warp distortion effects
---

<Steps>
<Step>
### Install the shader dependency

```npm
npm install @paper-design/shaders-react
```

</Step>
<Step>
### Add the component

```npm
npx shadcn@latest add https://glsilk.vercel.app/r/warp-button.json
```

</Step>
</Steps>

## Quick Start

```tsx title="app/page.tsx"
import { WarpButton } from "@/components/ui/warp-button";

export default function Home() {
  return (
    <div>
      <WarpButton>Warp Button</WarpButton>
    </div>
  );
}
```

<Callout type="info">
  **Default Colors:** The warp button now comes with a beautiful teal color
  scheme by default, featuring deep slate and vibrant teal tones that work well
  in both light and dark modes.
</Callout>

## Advanced Text Effects

Use the `textClassName` prop for custom text styling:

```tsx
<WarpButton textClassName="text-white mix-blend-difference">
  Inverted Text
</WarpButton>
```
