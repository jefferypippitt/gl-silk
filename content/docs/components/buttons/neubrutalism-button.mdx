### API Reference

The `NeubrutalismButton` component accepts all props that the standard shadcn/ui `Button` component accepts.

| Prop        | Type                                                                          | Default     | Description                                            |
| ----------- | ----------------------------------------------------------------------------- | ----------- | ------------------------------------------------------ |
| `variant`   | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"default"` | Button variant style.                                  |
| `size`      | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"`               | `"default"` | Button size variant.                                   |
| `className` | `string`                                                                      | —           | Additional CSS classes to apply to the button wrapper. |
| `children`  | `React.ReactNode`                                                             | —           | The content to display inside the button.              |
| `asChild`   | `boolean`                                                                     | `false`     | Render as a child element using Radix Slot.            |
| `...props`  | `ButtonProps`                                                                 | —           | All other props supported by the `Button` component.   |
