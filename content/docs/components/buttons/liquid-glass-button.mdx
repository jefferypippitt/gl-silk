### API Reference

| Prop                 | Type                                                                          | Default     | Description                                         |
| -------------------- | ----------------------------------------------------------------------------- | ----------- | --------------------------------------------------- |
| `distortionStrength` | `number`                                                                      | `0.15`      | Strength of the liquid distortion effect (0.05-0.3) |
| `borderRadius`       | `number`                                                                      | `0.6`       | Border radius for the glass effect (0.3-1.5)        |
| `animationSpeed`     | `number`                                                                      | `1`         | Speed of the animation (0.1-3)                      |
| `scale`              | `number`                                                                      | `1`         | Scale of the shader pattern (0.5-2)                 |
| `offsetX`            | `number`                                                                      | —           | Horizontal offset of the shader (-1 to 1)           |
| `offsetY`            | `number`                                                                      | —           | Vertical offset of the shader (-1 to 1)             |
| `rotation`           | `number`                                                                      | —           | Rotation angle of the shader (0-360)                |
| `worldWidth`         | `number`                                                                      | `0.5`       | Virtual width before scaling                        |
| `worldHeight`        | `number`                                                                      | `0.5`       | Virtual height before scaling                       |
| `textClassName`      | `string`                                                                      | —           | Custom className for the text content               |
| `variant`            | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"default"` | Button variant                                      |
| `size`               | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"`               | `"default"` | Button size variant                                 |
