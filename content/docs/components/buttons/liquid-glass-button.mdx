### Advanced Controls

Fine-tune the liquid glass effect with custom shader parameters. Use the interactive demo to experiment with different values.

<Tabs items={['Preview', 'Code']}>
<Tab value="Preview">
  <ComponentPreview name="liquid-glass-advanced" />
</Tab>
<Tab value="Code">

```tsx
<LiquidGlassButton
  distortionStrength={1.0}
  borderRadius={0.6}
  blur={0.25}
  brightness={1.5}
>
  Advanced Control
</LiquidGlassButton>
```

**Note:** The values above match the initial state of the interactive demo. The actual component defaults when no props are provided are:

- `distortionStrength={1.0}`
- `borderRadius={0.6}`
- `blur={0.25}`
- `brightness={1.2}` (demo uses `1.5` for better visibility)

</Tab>

### API Reference

| Prop                 | Type                                                            | Default                           | Description                                                                                                                                 |
| -------------------- | --------------------------------------------------------------- | --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| `distortionStrength` | `number`                                                        | `1.0`                             | Strength of the liquid distortion effect. Controls how much the glass effect bulges (0-2). Higher values create more pronounced distortion. |
| `borderRadius`       | `number`                                                        | `0.6`                             | Border radius for the glass effect shape. Controls the roundness of the distortion area (0-1.5).                                            |
| `blur`               | `number`                                                        | `0.25` (default), `1.0` (ghost)   | Backdrop blur intensity in pixels. Creates the frosted glass appearance (0-5).                                                              |
| `brightness`         | `number`                                                        | `1.2` (default), `1.0` (ghost)    | Backdrop brightness multiplier. Brightens or darkens the background content (0.5-2.5).                                                      |
| `textClassName`      | `string`                                                        | â€”                                 | Custom className for the text content. Useful for custom text colors or styling.                                                            |
| `variant`            | `"default" \| "ghost"`                                              | `"default"`                       | Button variant. Default has a semi-transparent background with the glass effect. Ghost variant has a transparent background with the glass effect always visible.                                                           |
| `size`               | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"` | `"default"`                       | Button size variant                                                                                                                         |

#### Examples

**Basic Usage**

```tsx
<LiquidGlassButton>Liquid Glass</LiquidGlassButton>
```

**Custom Shader Parameters**

```tsx
<LiquidGlassButton
  distortionStrength={1.5}
  borderRadius={0.8}
  blur={0.5}
  brightness={1.5}
>
  High Intensity
</LiquidGlassButton>
```

**Ghost Variant (Transparent Background)**

```tsx
<LiquidGlassButton variant="ghost">Minimal Style</LiquidGlassButton>
```

The ghost variant features a transparent background with the liquid glass effect always visible, perfect for minimal designs.

**Custom Text Color**

```tsx
<LiquidGlassButton textClassName="text-white mix-blend-difference">
  Inverted Text
</LiquidGlassButton>
```
