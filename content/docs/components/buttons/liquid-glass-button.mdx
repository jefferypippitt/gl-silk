- `distortionStrength={1.0}`
- `borderRadius={0.6}`
- `blur={0.25}`
- `brightness={1.2}` (demo uses `1.5` for better visibility)

### API Reference

| Prop                 | Type                                                            | Default                         | Description                                                                                                                                                       |
| -------------------- | --------------------------------------------------------------- | ------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `distortionStrength` | `number`                                                        | `1.0`                           | Strength of the liquid distortion effect. Controls how much the glass effect bulges (0-2). Higher values create more pronounced distortion.                       |
| `borderRadius`       | `number`                                                        | `0.6`                           | Border radius for the glass effect shape. Controls the roundness of the distortion area (0-1.5).                                                                  |
| `blur`               | `number`                                                        | `0.25` (default), `1.0` (ghost) | Backdrop blur intensity in pixels. Creates the frosted glass appearance (0-5).                                                                                    |
| `brightness`         | `number`                                                        | `1.2` (default), `1.0` (ghost)  | Backdrop brightness multiplier. Brightens or darkens the background content (0.5-2.5).                                                                            |
| `textClassName`      | `string`                                                        | â€”                               | Custom className for the text content. Useful for custom text colors or styling.                                                                                  |
| `variant`            | `"default" \| "ghost"`                                          | `"default"`                     | Button variant. Default has a semi-transparent background with the glass effect. Ghost variant has a transparent background with the glass effect always visible. |
| `size`               | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"` | `"default"`                     | Button size variant                                                                                                                                               |
