title: Dither Wave

---

## Installation

<Steps>
<Step>
### Install the shader dependency

```npm
npm install @paper-design/shaders-react
```

</Step>
<Step>
### Add the component

```npm
npx shadcn@latest add https://glsilk.vercel.app/r/dither-wave.json
```

</Step>
</Steps>

## Quick Start

```tsx title="app/page.tsx"
import { DitherWave } from "@/components/ui/dither-wave";

export default function Home() {
  return (
    <section className="h-[320px] overflow-hidden rounded-xl border">
      <DitherWave className="h-full w-full" />
    </section>
  );
}
```
