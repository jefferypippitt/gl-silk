**Material Types and Color Support:**

- `gradient`: Creates a smooth multi-color gradient using vertex colors (requires `gradientColors` prop, not `color`).
- `standard` (default): Supports single color customization with PBR lighting.
- `normal`: Supports color with specular highlights and phong shading.
- `basic`: Supports color but is unlit (flat appearance).
- `lambert`: Supports color with lambert shading (diffuse lighting).
- `toon`: Supports color with cartoon/toon shading effect.
- `physical`: Supports color with advanced PBR properties (metallic, roughness, etc.).

**Note**: All materials except `gradient` use the `color` prop. The `gradient` material uses `gradientColors` for multi-color effects.

### API Reference

| Prop                   | Type                                                                                   | Default            | Description                                                                                                                                                                                                         |
| ---------------------- | -------------------------------------------------------------------------------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `text`                 | `string`                                                                               | `"3D Text"`        | The text to display (use `\n` for line breaks)                                                                                                                                                                      |
| `size`                 | `number`                                                                               | `0.8`              | Font size                                                                                                                                                                                                           |
| `height`               | `number`                                                                               | `0.3`              | Extrusion height                                                                                                                                                                                                    |
| `bevelEnabled`         | `boolean`                                                                              | `true`             | Enable beveled edges                                                                                                                                                                                                |
| `bevelSize`            | `number`                                                                               | `0.02`             | Bevel size                                                                                                                                                                                                          |
| `bevelThickness`       | `number`                                                                               | `0.05`             | Bevel thickness                                                                                                                                                                                                     |
| `curveSegments`        | `number`                                                                               | `32`               | Curve segments for smooth curves                                                                                                                                                                                    |
| `letterSpacing`        | `number`                                                                               | `-0.06`            | Letter spacing                                                                                                                                                                                                      |
| `lineHeight`           | `number`                                                                               | `0.5`              | Line height                                                                                                                                                                                                         |
| `color`                | `string`                                                                               | `"#f5f5f5"`        | Text color (works with all materials except `gradient`). Use hex, rgb, or CSS variables. Optional - defaults to neutral gray.                                                                                       |
| `gradientColors`       | `string[]`                                                                             | `undefined`        | Array of color values for gradient effect (2, 3, 4, or more colors). Requires `materialType="gradient"`. Examples: `["#06b6d4", "#2196f3"]` (2 colors) or `["#06b6d4", "#2196f3", "#8b5cf6", "#ec4899"]` (4 colors) |
| `rotation`             | `[number, number, number]`                                                             | `[-0.5, -0.25, 0]` | Rotation in radians [x, y, z]                                                                                                                                                                                       |
| `position`             | `[number, number, number]`                                                             | `[0, 0, 0]`        | Position [x, y, z]                                                                                                                                                                                                  |
| `className`            | `string`                                                                               | `""`               | Additional CSS classes                                                                                                                                                                                              |
| `width`                | `number \| string`                                                                     | `"100%"`           | Container width                                                                                                                                                                                                     |
| `heightContainer`      | `number \| string`                                                                     | `400`              | Container height                                                                                                                                                                                                    |
| `materialType`         | `"standard" \| "normal" \| "basic" \| "gradient" \| "lambert" \| "toon" \| "physical"` | `"standard"`       | Material type. | `directionalIntensity` | `number`                                                                               | `0.8`              | Intensity of the directional light (0-2)                                                                                                                                                                            |

**Note on Height vs Bevel Thickness:**

Both `height` and `bevelThickness` control depth effects on the 3D text:

- `height`: Controls the extrusion depth (how far the text extends along the z-axis)
- `bevelThickness`: Controls the depth of the beveled edge (the angled surface connecting front and sides)

At certain scales and proportions, these parameters can appear to affect the text similarly because both add depth to the geometry. For a more pronounced distinction, try:

- Increasing `height` to 0.5-1.0 for more visible extrusion
- Keeping `bevelSize` and `bevelThickness` to smaller values (< 0.1) for subtle beveling
- Disabling `bevelEnabled` to see pure extrusion without bevel effects
