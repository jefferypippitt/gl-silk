All text animations require GSAP to be installed first. **No global CSS setup required** - the component is fully self-contained and inherits styles automatically.

<Steps>
<Step>
### Install GSAP

```npm
npm install gsap
```

</Step>
<Step>
### Add the component

```npm
npx shadcn@latest add https://glsilk.vercel.app/r/split-text-animation.json
```

</Step>
</Steps>

## Quick Start

```tsx title="app/page.tsx"
import { SplitTextAnimation } from "@/components/ui/split-text-animation";

export default function Home() {
  return (
    <SplitTextAnimation
      text="Hello World"
      animationType="fade"
      duration={0.8}
      stagger={0.03}
    />
  );
}
```

The component automatically inherits all font properties (size, weight, family, color) from its container. Apply any Tailwind classes or custom CSS to the component and all styles will be applied to each animated character:

<Tabs items={['Preview', 'Code']}>
<Tab value="Preview">
  <ComponentPreview name="split-text-custom-styling" />
</Tab>
<Tab value="Code">

```tsx
<SplitTextAnimation
  text="Custom Styled Text"
  className="text-2xl text-green-500"
  animationType="fade"
/>
```

</Tab>
</Tabs>

Supported CSS classes:

- **Text sizing:** `text-sm`, `text-lg`, `text-4xl`, etc.
- **Font weight:** `font-normal`, `font-bold`, `font-black`, etc.
- **Colors:** `text-blue-500`, `text-red-600`, etc.
- **Custom styles:** `font-mono`, `tracking-wider`, `italic`, etc.
- **Gradients:** `bg-gradient-to-r from-pink-500 to-violet-500 bg-clip-text text-transparent`

## API Reference

<TypeTable
  type={{
    text: {
      description: "The text to animate",
      type: "string",
    },
    animationType: {
      description: "Type of animation to apply",
      type: '"fade" | "slide" | "scale" | "rotate"',
      default: '"fade"',
    },
    direction: {
      description: "Direction for slide animation",
      type: '"up" | "down" | "left" | "right"',
      default: '"up"',
    },
    duration: {
      description: "Duration of the animation in seconds",
      type: "number",
      default: "0.8",
    },
    stagger: {
      description: "Delay between each character animation",
      type: "number",
      default: "0.05",
    },
    delay: {
      description: "Initial delay before animation starts",
      type: "number",
      default: "0",
    },
    className: {
      description: "Additional CSS classes (automatically inherited by all characters)",
      type: "string",
    },
    showRerunButton: {
      description: "Show a rerun button to replay the animation",
      type: "boolean",
      default: "false",
    },
  }}
/>
