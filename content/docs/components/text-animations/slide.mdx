### Usage

The command above will add the `SplitTextAnimation` component to your project. You can then use it like this:

```tsx {1,6} showLineNumbers title="app/page.tsx"
import { SplitTextAnimation } from "@/components/ui/split-text-animation";

export default function Home() {
  return (
    <div>
      <SplitTextAnimation
        text="Smooth Slide Up Animation"
        animationType="slide"
        direction="up"
        duration={0.8}
        stagger={0.03}
      />
    </div>
  );
}
```

### Props

| Prop        | Type                                  | Default  | Description                   |
| ----------- | ------------------------------------- | -------- | ----------------------------- |
| `text`      | `string`                              | required | The text to animate           |
| `direction` | `"up" \| "down" \| "left" \| "right"` | `"up"`   | Direction for slide animation |
| `duration`  | `number`                              | `0.8`    | Animation duration in seconds |
| `stagger`   | `number`                              | `0.05`   | Delay between each character  |
| `delay`     | `number`                              | `0`      | Initial delay                 |
