### Usage

The command above will add the `SplitTextAnimation` component to your project. You can then use it like this:

```tsx {1,6} showLineNumbers title="app/page.tsx"
import { SplitTextAnimation } from "@/components/ui/split-text-animation";

export default function Home() {
  return (
    <div>
      <SplitTextAnimation
        text="Dynamic Scale Animation"
        animationType="scale"
        duration={0.6}
        stagger={0.04}
      />
    </div>
  );
}
```

### Props

| Prop       | Type     | Default  | Description                   |
| ---------- | -------- | -------- | ----------------------------- |
| `text`     | `string` | required | The text to animate           |
| `duration` | `number` | `0.6`    | Animation duration in seconds |
| `stagger`  | `number` | `0.04`   | Delay between each character  |
| `delay`    | `number` | `0`      | Initial delay                 |
