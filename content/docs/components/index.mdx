### Component Categories

Glsilk provides a comprehensive collection of components organized by type:

- **[Buttons](/docs/components/buttons)** - Interactive button components with various styles
- **[Animations](/docs/components/animations)** - Smooth animations and transitions
- **[Text Animations](/docs/components/text-animations)** - Beautiful text animation effects
- **[Backgrounds](/docs/components/backgrounds)** - Beautiful background patterns and effects
- **[Special Effects](/docs/components/special-effects)** - Unique visual effects and interactions
- **[Device Mocks](/docs/components/device-mocks)** - Device mockups for showcasing your work
- **[Landing Pages](/docs/components/landing-page)** - Complete landing page sections
- **[Other](/docs/components/other)** - Miscellaneous components and utilities

### Installation

All components are available on the [Blocks page](/blocks) where you can preview and copy the code directly into your project.

### Customization

Every component is fully customizable. When you copy the code, it's yours to modify and extend as needed.
